The "MultiCanvas" Widget

This widget allows you to create several viewing canvases, each with a different visual type. Only one canvas is visible at any one time.

Library: karmawidgets
Link With: -lkarmawidgets

Tables

MultiCanvas_ResourceNamesList of resource names for MultiCanvas widget
MultiCanvas_ResourcesList of resources for MultiCanvas widget

Tables

MultiCanvas_ResourceNames List of resource names for MultiCanvas widget

Name Purpose
XkwNrequestMask Mask of requested canvas types to attempt to create
XkwNallowedMask Mask of allowed canvas types
XkwNsplitStereo If TRUE use a split stereo mode
XkwNverbose If TRUE verbose messages are displayed
