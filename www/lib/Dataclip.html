The "Dataclip" Widget

This widget provides an interactive way to select a range of data.

void XkwDataclipNewArray (Widget W, iarray array, double min, double max, flag reset_hscale)

Register new array.

Parameters:

  • W : The Dataclip widget.
  • array : The new array. This may be NULL.
  • min : The minimum (unscaled) data value in the array.
  • max : The maximum (unscaled) data value in the array. If this is less than min then the minimum and maximum values are computed.
  • reset_hscale : If TRUE the horizontal scale is reset.
Returns: Nothing.
Multithreading Level: Unsafe

EXPERIMENTAL FUNCTION: subject to change without notice

void XkwDataclipArrayChanged (Widget W, double min, double max)

Register a change in the array.

Parameters:

  • W : The Dataclip widget.
  • min : The minimum (unscaled) data value in the array.
  • max : The maximum (unscaled) data value in the array. If this is less than min then the minimum and maximum values are computed.
Returns: Nothing.
Multithreading Level: Unsafe

EXPERIMENTAL FUNCTION: subject to change without notice

void XkwDataclipSetRegions (Widget W, CONST DataclipRegions *regions)

Set the regions.

Parameters:

  • W : The Dataclip widget.
  • regions : The regions. These are unscaled (raw) values.
Returns: Nothing.
Multithreading Level: Unsafe
